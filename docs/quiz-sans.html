先想一下答案，再點開題目對照：

1. Codeberg為什麼決定在其使用條款中禁止主要由大型語言模型（LLM）生成的內容？

Codeberg禁止LLM生成內容主要是為了避免LLM生成代碼版權歸屬不確定所帶來的法律風險，這對於一個非營利組織來說是個嚴峻的問題。

2. 根據一項關於模型上下文協議（MCP）伺服器代理可用性的評估，即使是符合規範的伺服器，常見的問題是什麼？

研究發現，即使是符合規範的MCP伺服器，也常因缺乏良好的描述、命名不清晰及差勁的架構設計而無法被AI代理有效使用。特別是「未記錄參數」是一個嚴重問題。

3. 該研究指出，提升AI代理可用性的關鍵和最高槓桿點是什麼？

該研究指出，代理可用性關鍵在於「文件紀律」而非單純的工程問題。修正參數描述被認為是提升代理可靠性的最高槓桿點。

4. 根據美國國防部長Pete Hegseth的說法，與伊朗的戰爭迄今已花費美國多少錢？以及五角大廈請求額外撥款多少？

美國國防部長Pete Hegseth表示，與伊朗的戰爭迄今已耗費美國375億美元。五角大廈緊急向國會請求額外撥款870億美元，其中670億美元用於中東行動。
